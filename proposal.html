Puzzle Slider


Links:

  • Puzzle Slider Game
  • Documentation

Concept:

Puzzle Slider is a single player game in which you can do multiple puzzles in random order.
You can go from one puzzle to the next without having to finish the current one and can even
go back to the previous puzzle.


Genre:

Puzzle, Logic, Singleplayer


Platform:

Desktop only


Aesthetics:

Graphics style - Cartoon

Sound - 8-bit, or ambient

Sound types - Moving a tile of the puzzle, if they can't move the tile to a certain location,
and pressing buttons on the screen


Gameplay:

Mechanics:

Players can move around tiles next to each other by clicking and dragging,
then dropping the tile where they want it to go.

Control:

Mouse only

Teaching the game:

Instructions shown on the title screen and user can refer back to it as they play
the game by clicking the instructions button or can go back to the menu.

Player learning:

Players can create an image out of the tiles in randomized order to "win".
